# C++ Inventory Management System
A modular console-based Inventory Management System developed using C++. The application helps manage products, track stock levels, handle sales, and generate low-stock alerts using file-based persistence.

## Features
- Add new products to inventory
- Prevent duplicate product entries by merging quantities
- View all available products
- Update product price, quantity, and minimum stock level
- Sell products with automatic stock deduction
- Low-stock alert based on minimum stock threshold
- Delete products from inventory
- Persistent data storage using file handling

## Tech Stack
- Language: C++
- Concepts Used:
  - Object-Oriented Programming (OOP)
  - File Handling (ifstream, ofstream)
  - Functions and modular design
  - Conditional logic and loops
